# Netflix Data Analysis using Python & SQL Server

## Project Overview
This project demonstrates an end-to-end data pipeline:
- Raw Netflix data ingestion using Python (pyodbc)
- Data cleaning and transformation in SQL Server
- Advanced analytical queries using SQL

## Tools & Technologies
- Python (pandas, pyodbc)
- SQL Server
- Git & GitHub

## Key Analysis
- Director-wise movie & TV show counts
- Country-wise comedy movie analysis
- Yearly top directors
- Genre-based duration analysis
- Multi-genre director insights
